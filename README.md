# Algoritmos em Grafos — Atividade de Fixação

Este repositório contém as resoluções da atividade de fixação proposta na disciplina de Algoritmos em Grafos. Conforme as diretrizes e instruções do docente, todas as soluções foram desenvolvidas utilizando a linguagem C.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** C
* **Ambiente de Desenvolvimento:** Visual Studio Code (VS Code)
* **Controle de Versão:** Git / GitHub

## 📌 Escopo do Projeto
O objetivo principal é aplicar conceitos fundamentais da teoria dos grafos na prática, focando em eficiência, estruturação de dados (como matrizes e listas de adjacência) e implementação de algoritmos clássicos.
